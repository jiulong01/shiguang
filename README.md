# 拾光 (shiguang) 应用

拾光是一款基于 uni-app 框架开发的 纯前端跨平台移动应用 ，专注于任务管理、时间规划和效率提升。通过直观的界面和实用的功能，帮助用户更好地管理日常任务和时间。 本项目仅包含前端页面，无后端服务组件 。

## 主要功能

1. 1. 任务管理

   - 创建、编辑和删除任务
   - 任务分类和优先级设置
   - 拖拽排序功能

2. 2. 日历视图

   - 日程查看和管理
   - 日期选择和规划

3. 3. 数据统计

   - 任务完成情况统计
   - 时间利用效率分析

4. 4. 个人设置

   - 深色/亮色模式切换
   - 个性化偏好设置

5. 5. 会员系统

   - VIP 特权和订阅管理（前端模拟）

## 技术栈

- 前端框架 ：uni-app (支持 Vue 2 和 Vue 3)
- UI 组件 ：自定义组件 + uni_modules 组件库
- 动画效果 ：uni-transition
- 拖拽功能 ：vuedraggable
- 样式处理 ：Less
- 数据存储 ：uni.setStorageSync/uni.getStorageSync (本地存储)

## 纯前端特性说明

本项目是完全的前端实现，所有数据均存储在用户设备的本地存储中（部分功能还未完善）：

1. 1. 数据存储机制

   - 使用 uni-app 提供的 uni.setStorageSync 和 uni.getStorageSync API 进行本地数据存储
   - 数据仅保存在用户本地设备上，不会上传到任何服务器
   - 支持离线使用，无需网络连接即可访问和管理数据

2. 2. 数据持久化

   - 本地存储的数据在应用关闭后仍会保留
   - 清除应用缓存或卸载应用会导致数据丢失

3. 3. 无后端交互

   - 不涉及服务器请求和后端 API 调用
   - 所有功能均在前端完成，包括会员系统（模拟）和数据统计

## 安装与运行

1. 1. 环境要求

   - Node.js (推荐使用 LTS 版本)

   - HBuilderX (推荐) 或 VS Code + 相关插件

2. 2. 安装依赖

```
npm install
```

3. 1. 运行项目
   - 在 HBuilderX 中打开项目，选择运行目标（如微信小程序、浏览器等）
   - 或使用命令行运行：

```
# 运行到微信小程序
npm run dev:mp-weixin

# 运行到浏览器
npm run dev:h5
```

## 开发说明

- 项目支持 Vue 2 和 Vue 3，可通过 main.js 中的条件编译切换
- 使用 uni.navigateTo 进行页面跳转
- 使用 uni.setStorageSync/uni.getStorageSync 进行本地数据存储
- 组件间通信主要通过 props 和事件传递
- 由于无后端，所有数据处理逻辑均在前端完成

## 特别说明

本项目使用了 uni-app 框架，可实现一次开发，多端部署（微信小程序、支付宝小程序、H5 等）。所有数据均存储在本地，确保用户隐私安全，但也意味着数据不会在多设备间同步。
